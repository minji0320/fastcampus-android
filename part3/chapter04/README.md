# Part3 - Chapter04 도서 리뷰 앱

## 학습 내용
- RecyclerView
- View Binding
- Retrofit
- Glide
- Android Room
- Open API 활용

## 구현 내용
- 인터파크 Open API를 통해 베스트셀러 정보를 가져와서 화면에 그릴 수 있음
- 인터파크 Open API를 통해 검색어에 해당하는 책 목록을 가져와서 화면에 그릴 수 있음
- Local DB를 이용하여 검색 기록 저장, 삭제
- Local DB를 이용하여 개인 리뷰 저장
