# Part4 - Chapter01 유튜브

## 학습 내용
- ExoPlayer
  - custom controller
  - Playlist 등
- androidx.constraintlayout.widget.Group
- SeekBar custom

## 구현 내용
- Retrofit을 이용하여 재생 목록을 받아와 구성
- 재생 목록을 클릭 시, ExoPlayer를 이용하여 음악 재생 가능
- 이전, 다음 트랙 버튼을 눌러서 이전, 다음 음악 재생 & UI 업데이트
- PlayList 화면과 Player 화면 간의 전환
- SeekBar를 custom하여 원하는 UI로 표시
