# Part5 - Chapter07 영화 평점 서비스

## 학습 내용
- MVP
- Jetpack Navigation
- Koin
- Firebase Firestore

## 구현 내용
- 제공하는 영화 목록 조회
- 추천 영화 보여주기
- 내가 리뷰한 영화 목록 조회
- 영화 상세 화면에서 타인의 리뷰 확인 가능
- 영화 상세 화면에서 리뷰 작성 가능
