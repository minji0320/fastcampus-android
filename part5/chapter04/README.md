# Part5 - Chapter04 SNS 글 업로드 기능 개선하기

## 학습 내용

- Firebase :  Realtime Database, Storage
- Coroutines
- CameraX

## 구현 내용

- 기존 중고거래 앱 UI 수정 (SNS 컨셉)
- 사진 첨부 방식 두 가지 제공
    - 카메라 촬영 후 첨부
    - 갤러리 이미지 선택 후 첨부
- 이미지 리스트 첨부 및 여러개 동시 업로드
- 업로드 중 에러 발생 대응
