# Part2 - Chapter08 심플 웹 브라우저

## 학습 내용
- WebView
- WebViewClient, WebChromeClient
- SwipeRefreshLayout
- ContentLoadingProgressBar

## 구현 내용
- 주소창에 주소를 입력하여 웹 사이트 불러오기
- 뒤로 가기, 앞으로 가기 기능
- 홈 버튼을 눌러 처음 페이지로 이동
- 웹 사이트의 로딩 정도 확인 가능
